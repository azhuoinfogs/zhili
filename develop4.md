| **Server** | **S7** | 熔断降级 | 推荐接口熔断保护 | 后端 |

### 6.2 安全设计要点

**请求签名验证**：
```
签名 = SHA256(JSON(sorted(params)) + "&timestamp=" + timestamp + "&nonce=" + nonce + "&secret=" + APP_SECRET)
```

**敏感数据加密**：
- 用户手机号：AES-256 加密存储
- Token 过期时间：7-14 天

### 6.3 性能优化要点

**请求缓存策略**：
| 数据类型 | 缓存时间 | 缓存键 |
|----------|----------|--------|
| 商品详情 | 1小时 | `product:{id}` |
| 推荐列表 | 10分钟 | `recommend:{userId}:{profileId}:{hash}` |
| 转链结果 | 10分钟 | `purchase:{productId}:{platform}` |

**链路追踪**：
- 添加 `X-Trace-Id` 请求头
- 记录请求耗时、状态码、用户ID
